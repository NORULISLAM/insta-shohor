What's the differnce between local storage and session storage?

ANSWER:

What is sessionStorage?

sessionStorage is like a storage area in your browser, where you can store information for the duration of a web page session.
Information such as variables normally JavaScript. when reload the page then reset it. sessionStorage allows retain information when you close a page, as long as you don't close the browser window or tab.

What is LocalStorage?

Local Storage is a mechanism that allows user data to be saved in a web browser (local environment) by using javascript.
